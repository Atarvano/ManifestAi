Frequently Asked Questions

Jawaban untuk pertanyaan yang sering diajukan.

Apakah data saya aman?

Ya, sangat aman. Kami tidak menyimpan file yang Anda unggah secara permanen. Data diproses di memori server dan langsung dihapus setelah sesi berakhir.

Format file apa yang didukung?

Saat ini kami mendukung format Excel (.xlsx dan .xls). Pastikan file Anda memiliki header kolom yang jelas untuk hasil terbaik.

Apakah ini gratis?

Ya, Manifes Project saat ini tersedia gratis untuk penggunaan publik. Kami mungkin akan memperkenalkan fitur premium di masa depan.

Siap untuk memulai?

Tinggalkan cara manual yang lama. Beralih ke cara cerdas mengelola data manifes hari ini.
